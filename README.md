## 📊 Escenario Seleccionado: Escenario B – Análisis de Ventas de una Pequeña Empresa
El proyecto consiste en el procesamiento y análisis de un conjunto de datos comerciales simulados para extraer indicadores clave que permitan evaluar el desempeño de la organización.

### Objetivos del Análisis:
1. Importar el conjunto de datos en formato CSV.
2. Calcular indicadores comerciales fundamentales: Ventas totales, producto más vendido y evolución de ventas por mes.
3. Generar representaciones gráficas del rendimiento temporal.
4. Exportar y resguardar los resultados obtenidos de manera reproducible.
